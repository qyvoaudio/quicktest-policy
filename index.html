Resumo em 30 segundos — A extensão Quick Test funciona inteiramente dentro do seu navegador. Não coletamos, transmitimos ou vendemos seus dados. Não temos servidor. Tudo (áudios, perfis, configurações) fica salvo apenas no seu computador. A única vez que dados saem do navegador é quando você decide gerar um roteiro com IA — e nesse caso eles vão direto pra API que você escolheu, usando sua própria chave.

1.Quem somos

Quick Test é uma extensão para o navegador Google Chrome, desenvolvida e mantida por Qyvo Live.

Contato: contato@qyvolive.com.br

2.O que a extensão faz

A Quick Test ajuda criadoras e criadores de live commerce a manter um pitch de áudio contínuo durante transmissões ao vivo no TikTok Live Studio. Você grava frases curtas por categoria (gancho, oferta, escassez, prova social, etc.), a extensão monta combinações em loop e toca o áudio gerado, que pode ser roteado para a live via cabo virtual de áudio (BlackHole no Mac, VB-Cable no Windows).

3.Quais dados a extensão armazena

Tudo é salvo localmente no seu próprio computador, usando as APIs nativas do navegador. Nada é enviado para nossos servidores — porque não temos servidor.

Tipo de dadoOnde ficaPor quê
Arquivos de áudio que você sobeIndexedDB do navegadorPra tocar durante a transmissão
Perfis de produtoIndexedDB do navegadorPra alternar entre produtos

4.O que a extensão NÃO faz

  • Não tem servidor próprio
  • Não coleta telemetria (analytics, métricas de uso, crash reports)
  • Não compartilha seus dados com terceiros
  • Não vende seus dados
  • Não usa cookies de rastreamento
  • Não acessa páginas web que você visita
  • Não lê suas senhas, e-mails ou dados de outras abas

5.Quando dados saem do seu navegador

Há um único caminho: quando você decide usar o gerador de roteiro com IA dentro da extensão.

Nesse caso, a extensão envia o briefing que você preencheu para a API de LLM que você escolheu (Claude, OpenAI ou Gemini), usando sua própria chave. A extensão não intermedia esse envio — ele vai direto do seu navegador para o provedor da IA. O que acontece com esse dado depois é regido pelos termos do provedor que você escolheu:

  • Anthropic (Claude)
  • OpenAI
  • Google (Gemini)

Se você nunca usar o gerador de roteiro com IA, nada sai do seu navegador em momento algum.

7.Crianças

8.Seus direitos

30-second summary — The Quick Test extension runs entirely inside your browser. We do not collect, transmit, or sell your data. We do not operate a server. Everything (audio files, profiles, settings) is stored only on your computer. The only time data leaves your browser is when you choose to generate a script using AI — and in that case, it goes directly to the API you selected, using your own API key.

1.Who we are

Quick Test is a Google Chrome extension developed and maintained by Qyvo Live.

Contact: contato@qyvolive.com.br

2.What the extension does

Quick Test helps live commerce creators maintain a continuous audio pitch during TikTok Live Studio broadcasts. You record short phrases per category (hook, offer, scarcity, social proof, etc.), the extension assembles infinite loop combinations and plays the generated audio, which can be routed to the live stream via a virtual audio cable (BlackHole on Mac, VB-Cable on Windows).

3.What data the extension stores

Everything is saved locally on your own computer, using native browser APIs. Nothing is sent to our servers — because we don't have any.

Data typeWhere it livesWhy
Audio files you uploadBrowser IndexedDBTo play during broadcast
Product profilesBrowser IndexedDBTo switch between products
LLM API keys (optional)chrome.storage.localTo not retype them

4.What the extension does NOT do

  • No own server
  • No telemetry (analytics, usage metrics, crash reports)
  • No third-party data sharing
  • No data selling
  • No tracking cookies
  • No access to web pages you visit
  • No reading of passwords, emails, or other tab content

5.When data DOES leave your browser

There is only one path: when you choose to use the AI script generator inside the extension.

In that case, the extension sends the briefing you filled in to the LLM API you selected (Claude, OpenAI, or Gemini), using your own API key. The extension does not intermediate this request — it goes directly from your browser to the AI provider. What happens with that data afterwards is governed by the terms of the provider you chose:

  • Anthropic (Claude)
  • OpenAI
  • Google (Gemini)

If you never use the AI script generator, nothing leaves your browser at any point.
